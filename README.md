# Frutify

Frutify is a front-end web application for an online fruit store. It includes a homepage with a featured products carousel, an alphabetically organized catalog, a contact page with form validation, and a full login/registration system using localStorage. Built with pure HTML, CSS, and JavaScript — no frameworks required.
Features:

Responsive design with mobile-friendly layout
Product carousel with smooth scrolling
Client-side authentication (register, login, logout)
Contact form with toast notification feedback
Modal dialogs for "About Us" and "Terms of Service"
Scroll-triggered animations via CSS animation-timeline
